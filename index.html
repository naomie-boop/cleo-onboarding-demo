L'en-tête

Type · Loi · Pays · Pertinence (%) · Risque. En 2 secondes vous savez si c'est urgent.

Adapté à votre poste. DPO → angle données. Juriste → angle juridique. Même signal, résumés différents.

Vos produits croisés automatiquement avec le signal. Pas des catégories génériques.

⑥
Plan d'action

⑦
Contexte · Exposition · Escalade

Source · Feedback · Signet

Lien officiel · 👍👎 pour améliorer · 🔖 pour sauvegarder.

❓ FAQ

Combien de temps pour configurer ?
30 minutes. Profil, produits (import CSV), pays, premier scan.
Comment Cleo adapte les résumés ?
Département + Rôle dans Profil. Même signal, résumés différents selon votre poste.
Que sont les crédits ?
Chaque scan en consomme. Pro : 5 000/mois. Les crédits n'expirent pas.
D'où viennent les signaux ?
Registres fédéraux, journaux officiels, publications d'agences. Score de qualité dans Sources.
Cleo en équipe ?
Pro (5 membres) ou Enterprise (illimité). Chaque membre a ses résumés.
Alertes automatiques ?
Pro : alertes critiques, digest hebdo, rappels d'échéance. Slack disponible.
